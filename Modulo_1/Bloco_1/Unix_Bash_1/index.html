Introdução a Unix, Bash e Shell

Em meados dos anos 60, utilizando a linguagem de programação C. Ken Thopson e Dennis Ritchie, desenvolveram o esboço, do que um dia, seria o sistema operacional Unix, pai de quase todos os sistemas operacionais até hoje.

O sistema trouxe duas novidades que revolucionou o mundo da tecnologia para sempre:
- "Multi Usuarios" e "Multi Tarefas.

O Unix pode ser dividido em três niveis.

  • Kernel:
    Responsável pela interação entre hardwares e softwares.

  • Shell:
    É a interface entre o sistema operacional, o usuário e o Kernel.
    (Bash é um tipo de Shell)

  • Aplicativos:
    Basicamente, todos os softwares.

Definição on-line


Comando Iniciais no terminal

  • ctrl + alt + t
    Comando para abrir o terminal no Linux.

  • cd
    Comando para entrar, sair ou trocar de diretorio.

  • ls
    Comando para listar todos os arquivos e diretorios.

  • ls -a
    Comando para listar todos os arquivos e diretorios ocultos.

  • mkdir
    Comando para criar um diretorio.

  • rmdir
    Comando para remover um diretorio.

  • mv
    Comando para mover um arquivo ou diretorio.
    (mv nomeDoArquivoOuDiretorio /caminho/para/onde/mover).

  • rm
    Comando para remover um arquivo.

  • touch
    Comando para criar um arquivo.

  • clear
    Comando para limpar o terminal.

  • cat
    Comando para visualizar o conteudo de um arquivo.

  • pwd
    Comando para visualizar o caminho completo do root até o arquivo que você estiver.

  • cp
    Comando para copiar um arquivo.
    (cp nomeDoArquivo.txt nomeDoOutroArquivo.txt).

  • less
    Comando para visualizar o conteudo de um arquivo por páginas.
    ----> Dentro do modo de visualização, utilize:
    espaço para mudar de página.
    q para sair do modo de visualização.
    / para entrar no modo de busca.

  • head
    Comando para visualizar os dez primeiros conteudos de um arquivo.

  • head -5
    Comando para visualizar os cinco primeiros conteudos de um arquivo.

  • tail
    Comando para visualizar os dez ultimos conteudos de um arquivo.

  • tail -5
    Comando para visualizar os cinco ultimos conteudos de um arquivo.

  • grep
    Comando para buscar conteudos dentro um arquivo.
    (grep palavraDeBusca nomeDoArquivo.txt).
    Atenção, o grep é caseSensitive.
    Usar grep -i para buscar sem caseSensitive.
